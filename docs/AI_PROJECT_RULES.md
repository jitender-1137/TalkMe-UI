# AI Project Rules
This project uses Next.js App Router, TypeScript, Tailwind CSS, shadcn/ui,
TanStack Query, Zustand, Axios, React Hook Form, and Zod.

## Core Rules
- Follow existing architecture.
- No new patterns unless explicitly requested.
- Use named exports.
- Use strict TypeScript.
- No `any`.
- Reuse existing UI components.
- Reuse existing hooks and stores.
- Mobile-first responsive design.
- Production-ready code only.
